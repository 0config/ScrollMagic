ScrollScene

new ScrollScene(trigger, options)

CLASS ScrollScene (scene controller)

Parameters:
Name Type Argument Description
trigger string | object

The ScollScene object that is supposed to be updated.
options object <optional>

Options for the Scene. (Can be changed lateron)

Properties
Name Type Argument Default Description
duration number <optional>
 0

The duration of the scene. If 0 tweens will auto-play when reaching the trigger, pins will be pinned indefinetly starting at the trigger position.
offset number <optional>
 0

Offset Value for the Trigger Position
triggerPosition float | string | function <optional>
 "onEnter"

Can be string "onCenter", "onEnter", "onLeave" or float (0 - 1), 0 = onLeave, 1 = onEnter or a function (returning a value from 0 to 1)
reverse boolean <optional>
 true

Should the scene reverse, when scrolling up?
smoothTweening boolean <optional>
 false

Tweens Animation to the progress target instead of setting it. Requires a TimelineMax Object for tweening. Does not affect animations where duration==0
loglevel number <optional>
 2

Loglevel for debugging. 0: none | 1: errors | 2: errors,warnings | 3: errors,warnings,debuginfo

end

Scene end event. Fires whenever the scene's state goes from "DURING" to either "BEFORE" or "AFTER". Keep in Mind that it doesn't matter if the scene plays forward or backward: The event always fires when the scene leaves its active scroll timeframe, regardless of the scroll-direction.

Properties:
Name Type Description
event object

The event Object passed to each callback.

Properties
Name Type Description
name string

The unique name of the event.
newState string

Indicates towards which side we leave the scene: To the Top/Left (BEFORE) or Bottom/Right (AFTER)
